## 通信协议

### Vault 系统内部

```
浏览器 ──HTTP──▶ Next.js ──HTTP──▶ Vault Bridge ──SQLite──▶ vault.db
```

### 脱敏程序内部

```
React UI ──Tauri IPC──▶ Rust Backend ──sqlx──▶ vault.db
```

### 跨系统通信

```
Vault 系统 ──vault.db (文件系统)──▶ 脱敏程序
```

## 总结

这个架构的核心优势：

1. **简单** - 使用文件系统共享数据，无需复杂的网络通信
2. **安全** - 所有通信都在本地，不经过网络
3. **高效** - SQLite 查询速度快，UI 响应即时
4. **可靠** - 数据持久化在本地，不依赖外部服务
5. **易维护** - 模块化设计，职责清晰
