## 系统简介

**CheersAI Vault** 是一款专业的文件脱敏安全屋系统，帮助您安全地处理包含敏感信息的文件。

### 核心特性

- 🔒 **智能脱敏**：自动识别并脱敏身份证、手机号、邮箱等敏感信息
- 🔄 **可逆加密**：支持反脱敏，还原原始数据
- 📚 **自定义词库**：配置业务专属敏感词
- 🤖 **AI 增强**：可选 AI 多方法检测，提升识别准确度
- 📦 **多格式支持**：CSV、Excel、JSON、TXT、Word、PPT、PDF、Markdown
- 🔐 **安全沙箱**：加密存储敏感文件
- ☁️ **云端集成**：支持 FileBay 云存储

## 快速开始

### 安装

1. 下载 `CheersAI Desktop_x.x.x_x64-setup.exe`
2. 双击运行安装程序
3. 按照向导完成安装

### 首次使用

1. 启动 CheersAI Vault
2. 进入「增强服务」页面（可选）
   - 安装 OCR 服务（支持 PDF 文字识别）
   - 安装 AI 模型（支持智能检测）
3. 进入「敏感词库」页面
   - 添加您业务相关的敏感词
4. 开始使用文件脱敏功能

## 核心功能

### 1. 文件脱敏

将包含敏感信息的文件进行脱敏处理，保护数据安全。

#### 使用步骤

1. **进入文件脱敏页面**
   - 点击左侧菜单「文件脱敏」

2. **上传文件**
   - 或点击「点击选择文件」按钮
   - 支持批量上传

3. **选择脱敏规则**
   - 在右侧规则面板勾选需要的规则：
     - ✅ 身份证号
     - ✅ 手机号
     - ✅ 电子邮箱
     - ✅ 银行卡号
     - ✅ IPv4地址
     - ✅ 中文姓名
     - ✅ 护照号
   - **敏感词库**：自动启用（显示已配置的词条数量）

4. **配置输出选项**
   - **输出目录**：选择脱敏文件保存位置
   - **映射加密口令**：设置密码保护映射文件（可选）
     - 留空则不加密
     - 设置密码后反脱敏时需要输入

5. **AI 多方法检测**（可选）
   - 默认关闭（使用正则+敏感词库，速度快）
   - 开启后使用 AI+NER+正则+搜索四种方法
   - 适合需要高精度检测的场景

6. **预览脱敏效果**
   - 点击「开始处理」按钮
   - 系统自动加载预览
   - 查看脱敏前后对比

7. **确认并保存**
   - 检查预览结果
   - 点击「确认脱敏」保存文件
   - 生成两个文件：
     - `原文件名_masked.扩展名`：脱敏后的文件
     - `原文件名_masked.扩展名.cmap`：映射文件（用于反脱敏）

#### 支持的文件格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| CSV | `.csv` | 逗号分隔值文件 |
| Excel | `.xlsx`, `.xls` | Excel 工作簿 |
| JSON | `.json` | JSON 数据文件 |
| 文本 | `.txt` | 纯文本文件 |
| Word | `.docx` | Word 文档 |
| PowerPoint | `.pptx` | PPT 演示文稿 |
| PDF | `.pdf` | PDF 文档（需安装 OCR） |
| Markdown | `.md` | Markdown 文档 |

#### 脱敏规则说明

##### 内置规则

- **身份证号**：18位中国身份证号码
- **手机号**：11位中国手机号码
- **电子邮箱**：标准邮箱格式
- **银行卡号**：16-19位银行卡号
- **IPv4地址**：标准 IP 地址格式
- **中文姓名**：2-4个中文字符的姓名
- **护照号**：中国护照号码格式

##### 敏感词库

- 自动使用您在「敏感词库」页面配置的词条
- 支持精确匹配
- 支持中英文混合
- 按分类替换：`[分类名]`

##### 自定义正则规则

- 在「规则配置」页面添加
- 支持正则表达式
- 自定义替换模板

#### 脱敏示例

**原始数据：**
```
姓名：张三
身份证：110101199001011234
手机：13800138000
邮箱：zhangsan@example.com
公司：深圳科技有限公司
```

**脱敏后：**
```
姓名：[姓名_1]
身份证：[身份证_1]
手机：[手机_1]
邮箱：[邮箱_1]
公司：[地姓]科技有限公司
```

*注：假设"深圳"在敏感词库中，分类为"地姓"*

### 2. 文件反脱敏

使用映射文件还原脱敏后的数据。

#### 使用步骤

1. **进入文件反脱敏页面**
   - 点击左侧菜单「文件反脱敏」

2. **上传文件**
   - **脱敏文件**：选择 `*_masked.*` 文件
   - **映射文件**：选择对应的 `.cmap` 文件

3. **输入密码**（如果映射文件已加密）
   - 输入脱敏时设置的口令

4. **选择输出目录**
   - 选择还原文件的保存位置

5. **开始反脱敏**
   - 点击「开始反脱敏」按钮
   - 等待处理完成
   - 生成还原后的文件

#### 注意事项

- ⚠️ 映射文件必须与脱敏文件配对
- ⚠️ 如果映射文件加密，必须输入正确的密码
- ⚠️ 映射文件丢失或损坏将无法还原数据
- 💡 建议妥善保管映射文件

### 3. 敏感词库管理

配置您业务专属的敏感词，提升脱敏针对性。

#### 使用场景

- 公司名称、品牌名称
- 项目代号、产品名称
- 内部术语、专业词汇
- 地区名称、机构名称
- 其他业务相关敏感信息

#### 使用步骤

1. **进入敏感词库页面**
   - 点击左侧菜单「敏感词库」

2. **添加敏感词**
   - 点击右上角「+ 添加」按钮
   - 填写信息：
     - **敏感词**：要脱敏的词条（如"深圳"）
     - **分类**：词条分类（如"地姓"）
     - **描述**：备注说明（可选）
   - 点击「添加」保存

3. **批量导入**
   - 点击「导入」按钮
   - 选择 CSV 文件
   - CSV 格式：`分类,敏感词,描述,状态`
   - 示例：
     ```csv
     分类,敏感词,描述,状态
     地姓,深圳,城市名称,启用
     地姓,南山,区域名称,启用
     公司,腾讯,公司名称,启用
     ```

4. **管理敏感词**
   - **启用/禁用**：切换开关控制是否生效
   - **编辑**：修改词条信息
   - **删除**：移除词条
   - **搜索**：快速查找词条
   - **筛选**：按分类筛选

5. **导出敏感词**
   - 点击「导出」按钮
   - 选择保存位置
   - 生成 CSV 文件

#### 敏感词统计

页面顶部显示：
- **总词条数**：所有敏感词数量
- **已启用**：当前生效的词条数
- **已禁用**：未生效的词条数
- **分类数**：词条分类总数

#### 最佳实践

1. **合理分类**
   - 按业务领域分类（如：公司、项目、地区）
   - 便于管理和查找

2. **定期维护**
   - 及时添加新的敏感词
   - 清理过时的词条

3. **避免过度**
   - 不要添加过于通用的词（如"公司"、"项目"）
   - 控制词条数量（建议 < 1000）

4. **测试验证**
   - 添加后进行测试
   - 确保脱敏效果符合预期

### 4. 沙箱管理

安全存储和管理敏感文件。

#### 功能特点

- 🔐 **加密存储**：文件自动加密
- 🔑 **PIN 保护**：设置 PIN 码访问
- 📦 **导入导出**：支持文件打包
- 🔒 **锁定解锁**：控制文件访问

#### 使用步骤

1. **首次使用**
   - 进入「沙箱管理」页面
   - 设置 6 位 PIN 码
   - 记住 PIN 码（无法找回）

2. **上传文件到沙箱**
   - 拖拽文件到上传区域
   - 文件自动加密存储

3. **管理沙箱文件**
   - **查看**：列表显示所有文件
   - **下载**：导出文件到本地
   - **删除**：移除文件
   - **锁定**：禁止访问
   - **解锁**：允许访问

4. **导出沙箱**
   - 点击「导出沙箱」
   - 选择要导出的文件
   - 设置加密密码
   - 生成加密包

5. **导入沙箱**
   - 点击「导入沙箱」
   - 选择加密包文件
   - 输入密码
   - 导入到沙箱

#### 安全建议

- ⚠️ PIN 码无法找回，请妥善保管
- ⚠️ 导出密码要足够复杂
- 💡 定期备份重要文件
- 💡 不要在沙箱存储过大文件

### 5. FileBay 设置

配置 FileBay 云存储服务。

#### 功能说明

FileBay 是 CheersAI 的云存储服务，支持：
- 云端文件存储
- 多设备同步
- 团队协作
- 版本管理

#### 配置步骤

1. **获取配置文件**
   - 从 FileBay 管理后台下载 `filebay-config.json`
   - 或联系管理员获取

2. **导入配置**
   - 进入「FileBay 设置」页面
   - 点击「导入配置文件」
   - 选择 `filebay-config.json`
   - 系统自动验证并保存

3. **查看配置**
   - 显示当前配置状态
   - 查看服务器地址、端口等信息

4. **删除配置**
   - 点击「删除配置」
   - 断开 FileBay 连接

#### 配置文件格式

```json
{
  "server": "https://filebay.example.com",
  "port": 443,
  "apiKey": "your-api-key",
  "userId": "your-user-id"
}
```

### 6. 增强服务

安装和管理增强功能组件。

#### OCR 文字识别服务

**功能**：支持 PDF 文件的文字识别和脱敏

**使用场景**：
- 扫描版 PDF（图片型）

- 混合型 PDF

**安装步骤**：

1. 进入「增强服务」页面
2. 找到「OCR 文字识别服务」
3. 点击「一键安装」

5. 安装完成后可处理 PDF 文件

**技术说明**：
- 使用 PyMuPDF + EasyOCR
- 支持中英文识别
- 文本型 PDF：秒级提取
- 扫描版 PDF：30-60秒/页

**卸载**：
- 点击「完全卸载」
- 清除所有 OCR 组件

#### AI 模型服务

**功能**：提供 AI 智能检测能力

**使用场景**：
- 高精度姓名识别
- 复杂实体检测
- 多方法交叉验证

**安装步骤**：

1. **安装 Ollama**
   - 点击「下载 Ollama」按钮
   - 打开官网下载安装程序
   - 手动安装 Ollama

2. **安装 AI 模型**
   - Ollama 安装完成后
   - 点击「一键安装」
   - 等待下载模型（约 1GB）
   - 安装完成

**模型信息**：
- 模型：qwen2.5:1.5b
- 大小：约 1GB
- 用途：实体识别和验证

**卸载**：
- 点击「卸载模型」
- 保留 Ollama，仅删除模型

## 高级功能

### 自定义正则规则

在「规则配置」页面添加自定义脱敏规则。

**示例：脱敏工号**

1. 规则名称：`员工工号`
2. 正则表达式：`EMP\d{6}`
3. 替换模板：`[工号_{counter}]`
4. 启用计数器：✅

### AI 多方法检测

开启后使用四种方法检测敏感信息：

1. **正则表达式**：基于模式匹配
2. **NER（命名实体识别）**：机器学习模型
3. **AI 模型**：大语言模型推理
4. **搜索匹配**：敏感词库精确匹配

**检测策略**：
- **姓名**：四种方法交集（全部确认才脱敏）
- **其他**：四种方法并集（任一识别即脱敏）

### 批量处理

支持一次处理多个文件：

1. 拖拽多个文件到上传区域
2. 统一配置脱敏规则
3. 批量预览和确认
4. 自动生成所有脱敏文件

### 映射文件加密

保护映射文件安全：

1. 脱敏时设置口令
2. 映射文件自动加密
3. 反脱敏时需要输入口令
4. 支持 AES-256 加密

## 常见问题

### Q1: 脱敏后的文件可以还原吗？

**A:** 可以。只要保留映射文件（`.cmap`），就可以使用「文件反脱敏」功能还原。

### Q2: 映射文件丢失了怎么办？

**A:** 无法还原。映射文件包含脱敏前后的对应关系，丢失后数据无法恢复。建议：
- 妥善保管映射文件
- 定期备份
- 使用加密保护

### Q3: 为什么要配置敏感词库？

**A:** 内置规则只能识别通用敏感信息（身份证、手机号等），无法识别业务专属信息（公司名称、项目代号等）。配置敏感词库可以：
- 提升脱敏针对性
- 保护业务敏感信息
- 符合行业规范

### Q4: AI 检测和普通检测有什么区别？

**A:** 
- **普通检测**：使用正则+敏感词库，速度快，适合大部分场景
- **AI 检测**：增加 AI+NER 方法，精度高，但速度慢，适合高要求场景

### Q5: OCR 识别扫描版 PDF 需要多久？

**A:** 
- 首次使用：下载模型约 50MB，需要网络连接
- 识别速度：约 30-60 秒/页
- 14 页 PDF：约 7-14 分钟

### Q6: 支持哪些文件格式？

**A:** 
- **表格**：CSV, Excel (.xlsx, .xls)
- **文档**：Word (.docx), PDF, TXT, Markdown
- **数据**：JSON
- **演示**：PowerPoint (.pptx)

### Q7: 脱敏后的文件格式会变吗？

**A:** 不会。输出文件保持原格式，只是内容被脱敏。

### Q8: 可以自定义脱敏规则吗？

**A:** 可以。支持三种方式：
1. 选择内置规则
2. 配置敏感词库
3. 添加自定义正则规则

### Q9: 沙箱的 PIN 码忘记了怎么办？

**A:** 无法找回。PIN 码用于加密沙箱数据，忘记后只能清空沙箱重新设置。建议记录在安全的地方。

### Q10: 如何提高脱敏准确度？

**A:** 
1. 配置完善的敏感词库
2. 开启 AI 多方法检测
3. 使用预览功能检查效果
4. 根据业务调整规则
