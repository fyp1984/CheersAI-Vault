# 敏感词列表分页和排序功能

## 功能概述

敏感词列表现在支持分页显示、每页数量选择、搜索和多种排序方式。

## 新增功能

### 1. 分页显示

- **默认每页显示**: 5 条
- **可选每页数量**: 5条、10条、20条
- **分页导航**: 上一页、下一页、页码跳转
- **智能页码显示**: 当页数较多时，只显示当前页附近的页码

### 2. 每页数量选择

在操作栏中可以选择每页显示的数量：
- 每页 5 条（默认）
- 每页 10 条
- 每页 20 条

### 3. 排序功能

支持两种排序方式：
- **时间排序**: 按创建时间排序，最新的在前
- **首字母排序**: 按敏感词的首字母排序（支持中文拼音排序）

### 4. 搜索功能

- 实时搜索敏感词
- 按 Enter 键执行搜索
- 搜索结果也支持分页和排序

### 5. 分类筛选

- 按分类筛选敏感词
- 支持"全部分类"选项
- 筛选后自动重置到第一页

## 使用方法

### 切换每页数量

1. 在操作栏找到"每页 X 条"下拉框
2. 选择想要的每页数量（5、10 或 20）
3. 列表会自动更新并重置到第一页

### 切换排序方式

1. 在操作栏找到排序下拉框
2. 选择排序方式：
   - **时间排序**: 最新添加的敏感词在前
   - **首字母排序**: 按拼音字母顺序排列

### 分页导航

1. **上一页/下一页**: 点击箭头按钮
2. **跳转到指定页**: 点击页码数字
3. **页码显示规则**:
   - 总是显示第 1 页和最后一页
   - 显示当前页前后各 1 页
   - 中间用 "..." 表示省略的页码

### 搜索敏感词

1. 在搜索框输入关键词
2. 按 Enter 键或点击搜索按钮
3. 搜索结果支持分页和排序
4. 清空搜索框可恢复完整列表

## 技术实现

### 排序逻辑

```typescript
// 时间排序（新的在前）
if (sortBy === 'time') {
  sorted.sort((a, b) => {
    const dateA = new Date(a.created_at || 0).getTime();
    const dateB = new Date(b.created_at || 0).getTime();
    return dateB - dateA;
  });
}

// 首字母排序（支持中文）
else {
  sorted.sort((a, b) => {
    return a.term.localeCompare(b.term, 'zh-CN');
  });
}
```

### 分页逻辑

```typescript
const startIndex = (currentPage - 1) * pageSize;
const endIndex = startIndex + pageSize;
const pagedTerms = sorted.slice(startIndex, endIndex);
```

### 自动重置页码

当筛选条件变化时，自动重置到第一页：

```typescript
useEffect(() => {
  setCurrentPage(1);
}, [selectedCategory, searchQuery, pageSize]);
```

## 注意事项

### ⚠️ 重要提示

1. **分页重置**
   - 切换分类、搜索或改变每页数量时，会自动回到第一页
   - 这是为了避免显示空页面

2. **排序保持**
   - 排序方式在切换分类或搜索时保持不变
   - 可以在不同筛选条件下使用相同的排序

3. **搜索优先**
   - 搜索时会忽略分类筛选
   - 清空搜索后恢复分类筛选

4. **性能优化**
   - 使用 useMemo 优化排序和分页计算
   - 避免不必要的重新渲染

## 常见问题

### Q: 为什么切换分类后回到第一页？
A: 为了避免显示空页面。不同分类的数据量不同，自动重置到第一页确保有数据显示。

### Q: 首字母排序如何处理中文？
A: 使用 `localeCompare(term, 'zh-CN')` 方法，支持中文拼音排序。

### Q: 可以自定义每页数量吗？
A: 目前支持 5、10、20 三个选项。如需其他数量，可以在代码中添加。

### Q: 搜索时可以按分类筛选吗？
A: 搜索时会在所有敏感词中查找，不受分类限制。清空搜索后可以使用分类筛选。

### Q: 分页导航为什么有省略号？
A: 当页数较多时（超过 7 页），会用省略号代替中间的页码，保持界面简洁。
